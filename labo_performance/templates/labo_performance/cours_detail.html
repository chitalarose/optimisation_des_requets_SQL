Les bases des index

📚 7 leçons • ⏱ 30 min

Découvrez comment les index fonctionnent et comment ils peuvent considérablement améliorer les performances de vos requêtes SQL.

1. Qu'est-ce qu'un index ?

Un index est une structure de données qui améliore la vitesse de recherche dans une table. Sans index, la base doit scanner toutes les lignes (Full Table Scan). Avec un index, elle accède directement aux données.

2. Créer un index

Syntaxe de base :

CREATE INDEX nom_index ON nom_table (colonne);

Exemple :

CREATE INDEX idx_nom_utilisateur ON utilisateurs (nom);

3. Utiliser EXPLAIN pour vérifier

Vérifions que l'index est bien utilisé par le moteur :

EXPLAIN SELECT * FROM utilisateurs WHERE nom = 'John';
