Laboratoire : Avant / Après

Comparez les performances d'une requête non optimisée et d'une requête optimisée.

Requête non optimisée (Avant)

SQL 
SELECT * FROM commandes WHERE DATE(created_at) = '2024-01-01';

Temps d'exécution : 1250 ms

🙁

Lignes analysées : 523,000

Utilisation des index : ❌ Non

Requête optimisée (Après)

SQL 
SELECT * FROM commandes 
WHERE created_at >= '2024-01-01' 
AND created_at < '2024-01-02';

Temps d'exécution : 48 ms

😊

Lignes analysées : 1,253

Utilisation des index : Oui (idx_created_at)
